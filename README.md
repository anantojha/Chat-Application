# Chat-Application
Simple Chat Application using Socket Programming + Threads.
<br>
Supports Multiple Chat Clients.

<b>1.</b> Run ChatServer.java<br>
        ![RunServer](images/RunServer.png)

<b>2.</b> Run Multiple instances of ChatClient.java<br>
